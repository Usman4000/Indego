
# Description

Indego:ride sharing app. 

- use weahter api
- get-many (stations data)
- get-One (single station data)
- cron job
- static authentication   


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
